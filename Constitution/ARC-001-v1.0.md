## Примечание

Из-за ограничений размера ответа в чате полный документ (несколько десятков страниц)
не может быть целиком сформирован в одном сообщении.

Данный файл является стартовым артефактом для финальной редакции ARC-001 и
предназначен для дальнейшего наполнения без изменения структуры.
