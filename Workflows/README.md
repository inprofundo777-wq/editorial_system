Что такое Workflows

Это не инструкции отдельных редакторов.

Editors/Facebook/Workflow.md объясняет:

Как работает Facebook Editor внутри своей роли?

А Workflows/Article_Pipeline.md объясняет:

Как статья проходит через всю систему, переходя от одного редактора к другому?

То есть:

editor workflow — внутренний процесс одной роли;
system workflow — маршрут объекта через несколько ролей.
