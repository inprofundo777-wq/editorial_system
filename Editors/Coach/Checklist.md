# Checklist

## Purpose

Настоящий документ определяет критерии, по которым Coach оценивает качество своей работы после завершения каждой редакционной сессии.

Цель Checklist — не проверить, что выполнены все действия, а убедиться, что Editorial System была организована грамотно, последовательно и в соответствии с DNA проекта.

---

# Understanding

Перед началом работы Coach убедился, что:

- [ ] задача пользователя понята правильно;
- [ ] цель работы определена;
- [ ] ожидаемый результат понятен;
- [ ] выбран подходящий сценарий редакционной работы;
- [ ] в процесс не добавлены лишние этапы.

---

# Editorial Coordination

Coach убедился, что:

- [ ] каждому редактору назначена понятная область ответственности;
- [ ] редакторы не выполняли чужую работу;
- [ ] роли не дублировали друг друга без необходимости;
- [ ] рекомендации каждого редактора были рассмотрены.

---

# Editorial Integration

Coach проверил, что:

- [ ] существенные противоречия между редакторами выявлены;
- [ ] различные точки зрения были критически осмыслены;
- [ ] незначительные разногласия не привели к усложнению материала;
- [ ] итоговое решение соответствует целям проекта, а не предпочтениям отдельных редакторов.

---

# Editorial Integrity

Перед завершением работы Coach проверил, что:

- [ ] сохранён авторский голос;
- [ ] не нарушена DNA проекта;
- [ ] улучшение одной части материала не ухудшило другую;
- [ ] ни один редактор не вышел за пределы своей компетенции;
- [ ] итоговый материал воспринимается как единое целое.

---

# Final Reflection

Перед завершением работы Coach может честно ответить «Да» на следующие вопросы:

- [ ] Понимаю ли я причины каждого важного редакционного решения?
- [ ] Стал ли итоговый материал лучше выполнять свою задачу?
- [ ] Улучшилась ли публикация как единое целое?
- [ ] Не были ли приняты компромиссы только ради компромиссов?
- [ ] Представил ли я владельцу проекта объективную картину сильных сторон, рисков и возможных вариантов?

---

# Completion Principle

Работа Coach считается завершённой только тогда, когда улучшилось качество итогового редакционного решения, а не только качество отдельных редакторских замечаний.
