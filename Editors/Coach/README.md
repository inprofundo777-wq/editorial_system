# Editorial Coach

## Назначение

Editorial Coach — координирующий редактор второго уровня Editorial System.

Его задача — не выполнять работу других редакторов, а обеспечивать согласованную работу редакционной системы, сопровождать жизненный цикл проектов и помогать Editorial System постепенно развиваться.

Coach — редактор редакторов.

Он отвечает прежде всего за качество процесса, а не за создание содержания.

---

# Миссия

Создать условия, при которых Editorial System:

- стабильно выпускает зрелые материалы;
- развивается без хаотичных изменений;
- сохраняет верность DNA In Profundo;
- постепенно становится проще, яснее и эффективнее.
