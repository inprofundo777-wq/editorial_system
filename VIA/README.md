# VIA

VIA — единый рабочий путь идеи через Editorial System.

Каждая VIA:

- относится к одному создаваемому материалу или одной конкретной редакционной идее;
- хранится в одном документе;
- проходит через необходимые редакторские роли;
- сохраняет решения, передачи, изменения и итог публикации;
- после завершения остаётся в папке соответствующего года.

## Именование

`VIA-YYYY-NNN.md`

Пример:

`VIA-2026-001.md`

## Структура хранения

VIA сортируются по году открытия:

- `2026/`
- `2027/`
- `2028/`
